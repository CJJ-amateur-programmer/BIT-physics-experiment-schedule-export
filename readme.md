<h1>简介</h1>
本脚本针对北京理工大学物理实验选课系统，意在通过发送网络请求并处理返回的json数据，生成可以导入日历软件的.ics文件。<br>
<h1>使用说明</h1>
安装篡改猴等用户脚本管理器，将.js文件下载后导入，刷新网页后在篡改猴的菜单中找到“导出当前物理实验课表”的选项，点击即可。<br>
更详细的说明和注意事项请类比于：
